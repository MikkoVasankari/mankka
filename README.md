# mankka
Go CLI application to play media files with MPV.io

Requirements: <br>
[mpv.io](https://mpv.io/) <br>
[Go](https://go.dev/doc/install)

### To build project
To install project
``` 
.install/install.sh
```
and then to run use
```
mankka
```
