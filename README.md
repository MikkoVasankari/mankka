# mankka
Go CLI application to play media files with MPV.io

Requirements: <br>
[mpv.io](https://mpv.io/) <br>
[Go](https://go.dev/dl/)

### To build/run the project
To run use:
``` 
go run main.go
```
or build the executable

```
go build
```
